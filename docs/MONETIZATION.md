# ScriptFlow 商业模型设计
日期：2026-04-04
级别：生死线级别决策

## 核心原则
让用户先尝到天堂，再为进入天堂买票。
不能先把整部天堂电影院免费放完，再问他要不要结账。

## 绝对禁止
❌ 免费生成完整成片再收费
❌ 先生成再问用户要不要付钱
❌ 把主收费点放在导出环节

## 最终方案：方案D（混合预览+积分预付）

### 免费给用户
- 剧本预览
- 角色封面
- 1个低成本teaser片段

### 付费才触发（铁律）
- 完整多镜头视频生成
- 完整ElevenLabs配音
- 高清合成
- 无水印导出

## 积分消耗规则
1次完整短剧生成 = 3积分
1次teaser预览 = 免费
1次重新生成单镜头 = 0.5积分
完整重生成 = 2积分
高清无水印导出 = 1积分

## 定价
免费：注册送积分够1次teaser

Starter Pass：$9 = 3积分（1集完整）
Creator Pack：$29 = 12积分
Studio Pack：$59 = 30积分
导出无水印：+1积分或$2.99

## 冷启动策略
私密Beta招募（前30人）
手动赠送3次完整生成换取反馈
个人化1对1服务
建在公开制造FOMO
7-14天内拿到5个付费用户

## 长期进化
0-100用户：积分包为主
100-1000用户：月订阅$29
规模化：企业版+API
终极：生态分成
